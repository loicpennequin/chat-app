# Messenger-like application using React and nodejs

## Prerequisites:

-   nodeJS > 10.0.0
-   npm > 6.3.0
-   MySQL > 5.6

## Setup

-   `npm run setup:init`
-   wait for the promp in command lne to fill in your mySQL credentials
-   Open your browser of choice and go to http://localhost:8000
-   ...that's it.
